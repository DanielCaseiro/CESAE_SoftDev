Sobre mim



Olá! Meu nome é Daniel Caseiro, sou advogado e software developer, com mais de 5 anos de experiência em funções jurídicas, administrativas e de gestão.

Minha trajetória diversificada, que inclui atuações no Tribunal de Justiça de São Paulo, em escritórios de advocacia e em organizações sem fins lucrativos, me permitiu desenvolver sólidas habilidades jurídicas, de comunicação e de relacionamento interpessoal.

Recentemente, decidi direcionar minha carreira para a área de tecnologia, com foco em desenvolvimento de software. Para aprofundar meus conhecimentos, estou a concluir um curso de Software Developer na CESAE Digital. Com a base que já possuo em Java, HTML5, Bancos de Dados, SQL, Python, Analise de Dados, Power BI e Tableau, estou empenhado em aplicar minhas competências analíticas e de programação para impulsionar a inovação, resolver desafios de negócios e desenvolver soluções orientadas por dados.

Estas são algumas das minhas habilidades:

  • Java
  • Git
  • SQL
  • Python
  • HTML
  • CSS
